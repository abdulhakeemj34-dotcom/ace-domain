# Hosted Policy URL Map

Date: July 23, 2026.

Ace Domain already includes public in-app/web pages under `public/`. Store consoles need stable hosted HTTPS URLs before submission.

## Source Pages

- Privacy Policy source: `public/privacy/index.html`
- Terms and Safety source: `public/terms/index.html`
- Support source: `public/support/index.html`
- Account Deletion source: `public/account-deletion/index.html`

## Recommended Final URLs

- `https://acedomain.app/privacy/`
- `https://acedomain.app/terms/`
- `https://acedomain.app/support/`
- `https://acedomain.app/account-deletion/`

## Temporary Deployment Option

The repo includes `vercel.json`. A Vercel deployment can host the static pages and `/api/ai-chat`. If using Vercel temporarily, replace store-console URLs later with the permanent Ace Domain domain.

## Verification Checklist

- Each URL opens without login.
- Pages work on mobile and desktop.
- Support email is visible.
- Privacy page identifies app/operator and data practices.
- Account deletion page lets users start a deletion request.
- Terms/Safety page explains report/block/safety expectations.

Do not enter placeholder URLs in Play Console or App Store Connect.
