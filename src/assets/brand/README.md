# Ace Domain Brand Assets

Stage 10B creates the source-ready brand asset home for Ace Domain.

The approved direction comes from the attached Ace Domain brand board:

- Premium `AD` mark as the primary symbol.
- Electric blue, cyan, deep navy, black, and white palette.
- Rounded-square app icon treatment for Android and iOS readiness.
- Clean mobile splash/startup identity with light lines, logo assembly, brand name reveal, and glow settle.
- Tech-focused typography feel inspired by Space Grotesk, while the app continues to use the existing performant system stack for now.

No heavy bitmap assets are committed in Stage 10B. Future exported assets should be added here only after final approval and optimization.
