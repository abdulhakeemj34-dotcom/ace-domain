# Ace Domain Brand Assets

Stage 10B creates the source-ready brand asset home for Ace Domain. Stage 10D adds lightweight SVG masters for future native exports without replacing platform assets yet.

The approved direction comes from the attached Ace Domain brand board:

- Premium `AD` mark as the primary symbol.
- Electric blue, cyan, deep navy, black, and white palette.
- Rounded-square app icon treatment for Android and iOS readiness.
- Clean mobile splash/startup identity with light lines, logo assembly, brand name reveal, and glow settle.
- Tech-focused typography feel inspired by Space Grotesk, while the app continues to use the existing performant system stack for now.

No heavy bitmap assets are committed in Stage 10D. Future exported PNG assets should be added only after final approval, optimization, and device/emulator verification.

Current source-ready files:

- `ace-domain-mark.svg`
- `ace-domain-app-icon-master.svg`
- `ace-domain-splash-source.svg`
- `android-adaptive-foreground.svg`
- `android-adaptive-background.svg`

Native export notes live in `docs/native-brand-assets.md`.
