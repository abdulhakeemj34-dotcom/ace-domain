Account Deletion

Use this page to start an Ace Domain account deletion request and understand what happens next.

Account deletion removes the account record and associated personal data that Ace Domain is not legally required to keep.

Request Path

Send deletion requests to support@acedomain.app. Use the email address connected to the Ace Domain account when possible.

A deletion request should include the account email and a clear request to delete the Ace Domain account. Do not send passwords, recovery codes, API keys, or private tokens.

Start deletion request Email

Identity Verification

Ace Domain may verify account ownership without asking for passwords. Verification can use account email confirmation or another secure support workflow.

Data Reviewed For Deletion

  • Supabase auth user identity and profile records.
  • Remote user settings and local device settings guidance.
  • Posts, comments, community memberships, chat messages, and notifications when live.
  • Ace AI request logs if stored by the hosted backend.
  • Safety, fraud, moderation, or legal records that may require retention or anonymization.

Important Limits

Freezing, hiding, or logging out of an account is not the same as account deletion. Ace Domain explains what is deleted, retained, anonymized, and how long the request may take when processing the request.
